Welcome to The Memory Game

The game is simple. Four colours will appear on screen for 5 seconds. You must recall the four colours in the correct order as they appeared (from left to right) Once you fully recall all four colours a new set of colours will appear. The game ends when you get one wrong. How many can you get right.
